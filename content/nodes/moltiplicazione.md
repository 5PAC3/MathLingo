# Moltiplicazione

La moltiplicazione è un'**addizione ripetuta**. Moltiplicare $a$ per $b$ significa sommare $a$ con se stesso $b$ volte.

$$ a \times b = \underbrace{a + a + \cdots + a}_{b \text{ volte}} $$

**Proprietà:**
- **Commutativa**: $a \times b = b \times a$
- **Associativa**: $(a \times b) \times c = a \times (b \times c)$
- **Distributiva**: $a \times (b + c) = a \times b + a \times c$
- **Elemento neutro**: $a \times 1 = a$
- **Elemento assorbente**: $a \times 0 = 0$
