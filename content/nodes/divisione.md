# Divisione

La divisione è l'operazione inversa della moltiplicazione. Dividere $a$ per $b$ significa trovare il numero che moltiplicato per $b$ dà $a$.

$$ a \div b = c \quad \Longleftrightarrow \quad b \times c = a $$

$a$ si chiama **dividendo**, $b$ **divisore**, $c$ **quoziente**.

## Divisione esatta e con resto

Se $a$ non è un multiplo esatto di $b$, la divisione ha un **resto**:

$$ a \div b = c \text{ con resto } r, \quad \text{dove } 0 \leq r < b $$

**Verifica:** $a = b \times c + r$

**Importante:** Non si può dividere per zero!
