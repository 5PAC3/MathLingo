# Addizione

L'addizione è l'operazione che combina due o più numeri per ottenere una **somma**.

$$ a + b = c $$

**Proprietà:**
- **Commutativa**: $a + b = b + a$
- **Associativa**: $(a + b) + c = a + (b + c)$
- **Elemento neutro**: $a + 0 = a$

## Come si calcola

Allinea i numeri in colonna (unità sotto unità, decine sotto decine) e somma partendo da destra.
Quando la somma parziale supera 9, **fai il riporto** alla colonna successiva.
