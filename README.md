# MathLingo

App web open source stile Duolingo per la matematica. Dalle addizioni agli argomenti di quinta superiore, estendibile. Niente AI integrata, tutto generato proceduralmente.

## Stack

- **Frontend**: React/Next.js
- **Backend**: Python, FastAPI, SymPy, SQLite
- **Hosting**: GitHub Pages + Render
